# Gitpod Browser Setup - No Local Installation Required! 🚀

## Direct Browser Link

Open this link in your browser:
```
https://gitpod.io/#https://github.com/paritytech/DevEx-DevRel
```

## Important: Use Browser Mode

When Gitpod loads:
1. **If you see "Open in Desktop"** → Click **"Continue in Browser"** instead
2. **If VSCode opens locally** → Close it and go back to your browser
3. **If you see "Reopen in DevContainer"** → Close the prompt and ignore it

## What Happens Next

1. Gitpod will build the environment (first time takes ~5 minutes)
2. You'll be prompted to choose:
   - **Option 1**: Hardhat (JavaScript/TypeScript)
   - **Option 2**: Foundry (Rust/Solidity)
3. Enter a private key or press Enter to generate one
4. Start coding in your browser!

## Setting Gitpod to Always Use Browser

1. Go to: https://gitpod.io/user/preferences
2. Under "New Workspaces", select "Browser"
3. Save preferences
